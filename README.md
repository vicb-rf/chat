# ## Entrega 5.1: Xat

El ejercicio se divide en 2 partes el Backend y el Frontend.

Instalar los paquetes npm:

	>npm install

Backend:

Configurar parametros del servidor en el .env

Ejecutar 

	>npm run start

Frontend:

El servidor se ejecuta en el:

	>localhost:5000

Ejecutar 

	>npm run start
